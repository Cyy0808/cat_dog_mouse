输出汉诺塔步数:
输入:
汉诺塔的圈圈数n(1<=n<=20)
输出:
移动完所需的步数

样例输入0:
1
样例输出0:
1
说明:
只有一个圈圈,直接把他移动过去就是了

样例输入1:
2
样例输出1:
3
说明:
移动过程如下(|表示没有圈,数字表示圈的大小):
初始:
1  |  |
2  |  |
第一步:
|  |  |
2 1 |
第二步:
|  |  |
| 1 2
第三步:
| |  1
| |  2



进阶:
把移动过程也输出来,设三根柱子编号为1,2,3(默认初始圈圈都在1号柱子,要将他们都移动到3号),每次移动一个圈圈的时候,输出圈的起始柱子和目标柱子.例如
样例输入:
2
样例输出:
1 2
1 3
2 3
说明:
同上面最后一个例子
